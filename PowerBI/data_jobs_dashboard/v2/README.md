# Data Jobs Dashboard v2 with Power BI

![Dashboard Page 1](/images/Project2_Page1.png)

## Introduction

An enhanced version of the original Data Jobs Dashboard, developed to apply more advanced Power BI techniques and best practices in data visualization. This project explores key trends in the 2024 data job market, including salaries, job demand, and workforce insights, through a cleaner design, improved user experience, and more interactive reporting features.

The dashboard reflects the progression of my Power BI skills, showcasing stronger data modeling, visualization design, and analytical storytelling capabilities while delivering a more polished and professional reporting solution.

## Skills Showcased

* 🎨 **Dashboard Design:** Crafted a clean, modern, and intuitive layout focused on improving readability, user experience, and effective communication of insights.
* ⚙️ **Power Query ETL:** Extracted, transformed, and loaded raw data through Power Query to ensure data quality, consistency, and readiness for analysis.
* 🧩 **Data Modeling:** Built efficient relationships between tables and structured the data model to support accurate analysis, scalability, and reporting performance.
* 🧮 **DAX Fundamentals:** Applied core DAX functions and calculations to create meaningful measures, support business logic, and enhance analytical insights across the dashboard.
* **Visualizations Utilized:**
    * **📊 Core Charts:** Used cards and bar charts to visualize salary trends, skill demand, and job market insights in a clear and engaging way.
    * **🃏 Cards:** Designed dynamic KPI cards to highlight key metrics such as job count, median yearly salary, median hourly salary, and average skills per job, providing an instant overview of the data job market.
    * **📈 Chart Variety:** Combined cards, bar charts, slicers, and toggle buttons to present data from multiple perspectives, improving analysis and user engagement throughout the dashboard.
* **🖱️ Interactive Features:** Enhanced user exploration through slicers, parameters, and a custom clear filters button, enabling flexible and personalized analysis of the data.
