# Data Analysis Projects

A collection of data analysis projects covering SQL, Power BI, Python, and Excel.

## About

Chemical Engineer working on data analysis, business intelligence, and process optimization. This repository brings together hands on projects built to practice and demonstrate skills in querying, visualizing, and interpreting data.

## Repository structure

```
Data-Analysis-Projects/
├── SQL/          Query based projects and exercises
├── PowerBI/      Interactive dashboards and reports
├── Python/       Data analysis and scripts
└── Excel/        Models and analysis in Excel
```

Each folder contains independent projects, each with its own README describing the objective, process, and results.

A couple of projects use more than one tool (e.g. **sales_analytics_pipeline**). For those, each tool's part lives in its own folder under the same project name, with its own README linking to the other parts.
