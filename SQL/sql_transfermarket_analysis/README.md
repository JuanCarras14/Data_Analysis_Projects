# Football Transfer Market Analysis

## Project Overview

This project explores football transfer market data using SQL to answer business questions related to player valuation, clubs, countries, and market trends.

The project is organized as a real-world analytics workflow:

1. Data Exploration
2. Data Quality Assessment
3. Business Analysis

## Dataset

Transfermarkt Football Dataset

## Tools

- SQLite
- DB Browser for SQLite
- Git
- GitHub
